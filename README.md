# SafeHaven - How to Edit
simple html
---
## Change Text
Open `index.html` in any text editor (Notepad, VS Code, etc).
### Hero Section
Find and edit:
- `"You can talk to us."` - main headline
- `"If something bad is happening online..."` - subtext
- `"Lets talk"` / `"See tools"` - buttons
### Chat Bot Message
Find and change:
- `"hey, its Julian. i built this chat bot thing..."` - welcome message
- `"yeah i hear you. that sounds rough..."` - response
### Tool Headers
Find and change titles like:
- `"What to do right now"`
- `"How to block or report someone"`  
- etc.
### Tool Items
Edit the text inside the tool sections.
### Resources Cards
Find each card and edit:
- Title
- Description text
### Footer
Edit:
- `"Privacy"`, `"Terms"`, `"Contact"` links
- `"SafeHaven 2026"`
---
## That's It
Just save and upload to GitHub. Done.
